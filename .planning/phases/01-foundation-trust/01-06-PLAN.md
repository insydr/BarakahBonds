# Plan 01-06: Fix Button asChild Prop Issue

**Problem:** The Button component uses `@base-ui/react/button` which does not support the `asChild` prop pattern. Multiple components throughout the codebase use `<Button asChild>` expecting Radix Slot behavior, causing TypeScript errors.

**Solution:** Add Slot support to the Button component using `@radix-ui/react-slot`, enabling the `asChild` pattern for composing with Link and other elements.

## Files Affected

|------|-------|

| `app/(protected)/settings/page.tsx` | 1 occurrence of `<Button asChild>` |
| `app/(protected)/dashboard/page.tsx` | 3 occurrences of `<Button asChild>` |

export interface ButtonProps
